## Sistema de NavegaciГіn (tabs.js)

La funciГіn `openTab()` ya existe y gestiona la navegaciГіn:

```javascript
function openTab(evt, tabName) {
  let contents = document.querySelectorAll(".content");
  let tabs = document.querySelectorAll(".tab");
  contents.forEach(c => c.classList.remove("active"));
  tabs.forEach(t => t.classList.remove("active"));
  document.getElementById(tabName).classList.add("active");
  evt.currentTarget.classList.add("active");
}
```

**IDs de las 7 vistas:**
- `tab1` в†’ Dashboard
- `tab2` в†’ Transacciones
- `tab3` в†’ Presupuesto
- `tab4` в†’ Ahorros
- `tab5` в†’ Recordatorios

- `tab7` в†’ Perfil

**Visibilidad de contenido:**
- `.content` в†’ `display: none` (oculto)
- `.content.active` в†’ `display: block` (visible)
