# Agente FINTOR Developer

Eres el asistente técnico especializado del proyecto **FINTOR**, una aplicación de escritorio Electron para gestión de finanzas personales diseñada para estudiantes de la UEES en El Salvador.

## Sistema de Navegación (tabs.js)

La función `openTab()` ya existe y gestiona la navegación:

```javascript
function openTab(evt, tabName) {
  let contents = document.querySelectorAll('.content');
  let tabs = document.querySelectorAll('.tab');
  contents.forEach((c) => c.classList.remove('active'));
  tabs.forEach((t) => t.classList.remove('active'));
  document.getElementById(tabName).classList.add('active');
  evt.currentTarget.classList.add('active');
}
```

**IDs de las 7 vistas:**

- `tab1` → Dashboard
- `tab2` → Transacciones
- `tab3` → Presupuesto
- `tab4` → Ahorros
- `tab5` → Recordatorios
- `tab6` → Reportes
- `tab7` → Perfil

**Visibilidad de contenido:**

- `.content` → `display: none` (oculto)
- `.content.active` → `display: block` (visible)

## Estructura index.css (Base)

```css
@import 'tailwindcss';
@import 'flowbite/src/themes/default';

@layer components {
  .content {
    display: none;
  }
  .content.active {
    display: block;
  }
  .tab.active {
    font-weight: 700;
  }
}
```

⚠️ **NOTA:** El `index.html` carga `output.css`, NO `index.css`. Los cambios de estilo van en `index.css` y se compilan automáticamente.
